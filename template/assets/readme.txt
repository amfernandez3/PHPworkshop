Esqueleto de proyecto con la estructura básica de un CRUD en PHP:
Dividido en:
- modelo: Modelos de las clases y estructuras que se usan en el proyecto.
- vista: Vistas a las que podrá acceder el usuario que interactúe con el servidor.
- controlador: Control de interacción y DB
- DB: Contenidos referentes a las bases de datos del proyecto.
- assets: directorio para recursos multimedia.
